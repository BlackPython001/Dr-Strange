# 🧙 Doctor Strange – Mystic Arts Engine

> A cinematic Doctor Strange inspired hand-tracking magic engine built entirely in JavaScript using MediaPipe Hands and HTML5 Canvas.

## ✨ Features

- 🛡 Mystic Energy Shield
- 🌀 Sling Ring Portal
- ☄ Finger Energy Trails
- 🌫 Mystic Smoke Effects
- ⚡ Magic Sparks
- ⭐ Animated Starfield Background
- 🖐 Real-Time Hand Tracking
- 🎨 Smooth Particle Animation
- 💻 Runs completely in the browser
- 📷 Webcam powered

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Doctor-Strange-Mystic-Arts.git
```

Go inside the folder

```bash
cd Doctor-Strange-Mystic-Arts
```

Open

```
index.html
```

or run with

```bash
python -m http.server
```

and visit

```
http://localhost:8000
```
